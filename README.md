# E-Buy

Progetto per l'esame di ingegneria del software.

#### Prerequisiti

1. cmake
2. make (opzionale)
3. un compilatore cpp (clang, gcc, ecc.)
4. git
5. la libreria PostgreSQL
6. libreria hiredis


#### Primi Passi

```sh
git clone git@github.com:n4y0n/progetto-is-2024.git --recurse
```
```sh
cd progetto-is-2024
```
```sh
make
```
```sh
# cmake -S . -B build
```
```sh
# cmake --build build
```
