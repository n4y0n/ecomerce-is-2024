# E-Buy

Progetto per l'esame di ingegneria del software.

#### Prerequisiti

1. la libreria PostgreSQL
2. cmake
3. make (opzionale)
4. un compilatore cpp (clang, gcc, ecc.)
5. git


#### Primi Passi

```sh
git clone git@github.com:n4y0n/progetto-is-2024.git --recurse
```
```sh
cd progetto-is-2024
```
```sh
make
```
```sh
# cmake -S . -B build
```
```sh
# cmake --build build
```
